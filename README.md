\# Satellite Home Watch App



Frontend web application for \*\*Satellite Home Watch\*\*, a property monitoring service.



This project was generated with AI tools and cleaned for production export before backend integration.



\---



\## Current State



The project currently contains:



\- Fully built frontend UI

\- Admin dashboard

\- Client dashboard

\- Employee dashboard

\- CRM pages

\- Messaging system UI

\- Visit tracking UI

\- Reports and documents UI

\- Mobile responsive landing page

\- Progressive Web App (installable mobile app)



All mock data, demo content, and placeholder logic have been removed.



\---



\## Tech Stack



Frontend:



\- HTML

\- CSS

\- Vanilla JavaScript

\- Progressive Web App support



Planned Backend:



\- Supabase (database + authentication)

\- Cloudflare hosting

\- GitHub repository



\---



\## User Roles



The system supports three main roles:



\*\*Admin\*\*

\- manage clients

\- manage properties

\- view analytics

\- manage visits

\- manage employees

\- manage reports



\*\*Client\*\*

\- view property status

\- receive alerts

\- view inspection reports

\- track visits

\- message the team



\*\*Employee\*\*

\- perform property inspections

\- log visits

\- upload reports

\- communicate with admins
